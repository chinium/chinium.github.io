FIGlet - display large characters made up of ordinary screen characters  
URL: <http://www.figlet.org/>{:target="_blank"}

### install
```bash
[root@my-dev temp]# yum install figlet
```


### example
```bash
[root@my-dev temp]# figlet Hello world
```
